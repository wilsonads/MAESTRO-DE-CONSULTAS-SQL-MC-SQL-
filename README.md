# MAESTRO-DE-CONSULTAS-SQL-MC-SQL-
El Programa Maestro de Consultas es una herramienta de escritorio segura y eficiente para bases de datos SQL. Facilita la extracción y el análisis de datos con un editor inteligente, pero bloquea los comandos de modificación , garantizando la integridad de la base de datos. Permite gestionar consultas, guardar conexiones y exportar resultados.
